#1. PPE - Serveur AD

Contexte : Durant la seconde année de ma formation, nous avons eu à installer et gérer un serveur AD sous Windows Server.

Objectif : Installer une machine Windows Server et un service active-directory qui permet la gestion d'utilisateur, de groupes, et de GPO.

Travail réalisé : Un serveur bi-cpu avec 256Gb de Ram permettant de vitualiser des machines et services. Sur celui-ci il y a : une VM (AD/Domaine, DHCP, DNS, Fichier/DFS et servant a des outils internes) une VM pour un controler wifi, une VM de base de donné, une VM pour la sauvgarde, une VM pour un serveur WEB.. Le tout fonctionnait sous Windows Serveur 2022 hôtes comme Guest sauf le controler wifi sur base debian.

Technologies utilisées : Windows Serveur 2022, Hyper-V, Debian, Apache, SQL Server, Windows AD, Veeam

Compétances mobilisées :
C1: Gérer le patrimoine informatique
C3: Développer la présence en ligne de l'oganisation
C4: Travailler en mode projet
C5: Mettre a dispostion des utilisateurs un service informatique

Resultats obtenus : 3 mois plus tard nous avons obtenu un livrable prêt a l'utilisation pour les clients

Difficultés rencontrées : Le timing était complexe du a l'alternance (j'ai eu 1 moi et demi au lieu de 3 mois)

#2. PPE - Serveur Web

Contexte : Durant la seconde année de ma formation, nous avons eu à installer et configurer un serveur Web sous Linux.

Objectif : Installer une machine Linux et un apache2 ou nginx qui permet d'hébérger une/des page(s) web sur une machine.

Travail réalisé : Installer une machine virtuel sous ubuntu serveur 24.04 LTS (long-term support, 5 ans de majs gratuites jusqu'a 15 ans avec l'abonnement) et apache2. Sur celle-ci, une fois le service installé les certificats ssl afin que les pages web s'affiches en https.

Technologies utilisées : Linux server 24.04 LTS, Apache et OpenSSL

Compétances mobilisées :
C1: Gérer le patrimoine informatique
C3: Développer la présence en ligne de l'oganisation
C6: Organiser son dévloppement professionnel

Resultats obtenus : Un serveur Web Linux/apache hébérgeant

Difficultés rencontrées : les repos apt ne fonctionnent pas toujours
